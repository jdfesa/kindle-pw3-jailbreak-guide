# Experimentos de personalización

Esta carpeta está separada de `docs/` para que la guía de jailbreak siga siendo reproducible y no se confunda con software propio aún experimental.

## Objetivo

Investigar y, en una etapa posterior, construir una pequeña aplicación para el Kindle Paperwhite 3 de prueba. El nombre de trabajo es **PW3 Dashboard**; no es todavía un producto ni una instalación recomendada.

## Documentos

- [FEASIBILITY.md](FEASIBILITY.md): qué es viable, qué requiere pruebas y qué queda fuera de alcance.
- [ARCHITECTURE.md](ARCHITECTURE.md): tecnologías comparadas y arquitectura inicial recomendada.
- [ROADMAP.md](ROADMAP.md): fases, comprobaciones y criterios para no poner en riesgo el lector.
- [BACKLOG.md](BACKLOG.md): tareas pendientes, prioridades, criterios de aceptación y orden de ejecución para la próxima sesión.

## Regla de separación

Nada de esta carpeta es necesario para WinterBreak2, Hotfix, KUAL, Rename OTA Binaries, KOReader ni KindleForge. Cualquier prototipo debe poder retirarse borrando únicamente su carpeta y su lanzador, sin tocar la partición del sistema.
