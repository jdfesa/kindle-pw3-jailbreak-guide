# Fondos para el Kindle Paperwhite 3

## `library-mountain-pw3.png`

Ilustración original generada para este proyecto con la herramienta integrada de
generación de imágenes. Se conserva en el repositorio para poder reinstalarla o
adaptarla sin depender del estado actual del Kindle.

Especificaciones finales:

| Propiedad | Valor |
|---|---|
| Dimensiones | `1072 × 1448` |
| Color | Escala de grises |
| Niveles | 16, con dithering Floyd–Steinberg |
| Profundidad PNG | 8 bits grayscale |
| Destino | Kindle Paperwhite 3, pantalla de 300 ppp |

Prompt final:

> Create an elegant literary monochrome illustration: a lone reader seen from
> behind at the entrance of an immense quiet library carved into a dark
> mountain, with shelves and stairways fading upward into light;
> contemplative, timeless, slightly mysterious. Detailed black ink engraving
> with restrained charcoal shading, fully original, strict portrait
> composition, strong black/white contrast, no text, logos, watermark, border
> or UI.

El original generado era RGB de `1080 × 1456`. La versión publicada se recortó
al centro, se convirtió a escala de grises y se cuantizó con ImageMagick. No
contiene texto ni marcas de terceros.
