# Videos de verificación

Este directorio conserva evidencia visual pública de la ejecución real. No debe
contener números de serie completos, redes Wi-Fi, credenciales, conversaciones
ni otros datos personales.

## Arranque automático de KOReader

- `kindle-pw3-arranque-koreader.mp4`: reinicio normal del PW3 hasta la Home de
  KOReader; 44,68 segundos, H.264 Main, `740 × 1080`, píxeles cuadrados, carga
  progresiva y sin audio ambiente.
- `kindle-pw3-arranque-koreader-poster.jpg`: fotograma usado como presentación
  centrada y enlace estable desde el README principal.

El MP4 sustituye un adjunto temporal de GitHub que no terminaba de procesarse.
Versionarlo junto con la documentación permite verificar su integridad y evita
depender de una URL anónima de `user-attachments`.
